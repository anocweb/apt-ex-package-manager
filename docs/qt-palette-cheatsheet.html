Usage in Qt Stylesheets

Use these palette colors in your stylesheets to ensure theme compatibility:

/* Sidebar with darker background */
QWidget#sidebar {
  background-color: palette(base);
  border-right: 1px solid palette(mid);
}
/* Button with theme-aware colors */
QPushButton {
  background-color: palette(button);
  color: palette(window-text);
}
QPushButton:hover {
  background-color: palette(alternate-base);
}
/* Selected state */
QPushButton[selected="true"] {
  background-color: palette(highlight);
  color: palette(highlighted-text);
}

Common Use Cases

  • Window - Main window background
  • Base - Darker panels, sidebar (dark theme: #141618)
  • AlternateBase - Hover states, alternating rows
  • Button - Button backgrounds
  • Mid - Borders, separators
  • Highlight - Selected items, active states
  • HighlightedText - Text on highlighted backgrounds
